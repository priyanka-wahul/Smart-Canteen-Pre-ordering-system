## Smart Canteen Pre-ordering System

## A full-stack food ordering web application where users can:

- Signup/Login
- Browse menu items
- Add items to cart
- Delete cart items
- Checkout with payment option
- Place orders
- View flash messages
- Session-based authentication

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
- Passport.js
- Express Session

## Features

- User Authentication
- Cart Management
- Order Placement
- Flash Messages
- Error Handling
- MVC Architecture
