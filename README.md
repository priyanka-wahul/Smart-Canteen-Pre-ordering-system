## This is college hackathon project.

## Smart Canteen Pre-ordering System

## A full-stack food ordering web application where users can:

- Signup/Login
- explore menu items
- Add items to cart
- Delete cart items
- Checkout with payment option
- Place orders
- View flash messages
- Session-based authentication
